# CartSavvy Browser Extension

CartSavvy is an intelligent shopping assistant that helps you optimize your cart across multiple vendors to find the best deals and maximize savings.

## Getting Started

1. **Visit the Web App**: Go to [cartsavvy.vercel.app](https://cartsavvy.vercel.app) to manage your carts and view optimization results.
2. **Install the Extension**: Download the `cartsavvy-extension.zip` from this repository and load it into your browser (Chrome/Edge).
3. **Usage Guide**: For detailed instructions on how to set up and use CartSavvy, please visit our [How to Use Guide](https://cartsavvy.vercel.app/how-to-use.html).

## Features

- **Multi-Vendor Optimization**: Automatically compares items across vendors like Amazon, Blinkit, Zepto, and more.
- **Cashback Awareness**: Accounts for credit card rewards and cashback offers.
- **Smart Item Matching**: Uses AI to ensure you're comparing identical or better-value items.
